◆ Attack()

void Scripts.Actions.PlayerAttack.Attack ( )

This is the base funciton for all attacks.

The trigger for this function to be used is a button on the UI canvas. Once the button is triggered the player must have already selected a figure to attack or it will get an error.

This function will take away any health if need be or it can even trigger the destruction of an object.
