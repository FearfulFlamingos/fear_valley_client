This is a networking function used when other players call a retreat in the game.

Parameters
troopIdTroop retreating
TeamNumThe team that is leaving

See Leave

This function is checked each time a character dies. It removes them from the internal team dictionaries, then checks if the endgame has been triggered.

This used to have a separate set of checks for if the enemy/player retreated thier last troop, but I think it's unnecessary.

troopIdID of the troop.
teamTeam that the character was removed from.
