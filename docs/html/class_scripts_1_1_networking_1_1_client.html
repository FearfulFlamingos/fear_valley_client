Detailed Description

Attaches to a client object and handles all communication with the server.

There are a lot of precompiler options being used. This is to disable the warnings about how Unity's netcode is being deprecated. Since the replacement has yet to come out and the code is still valid in this version of unity, it's cleaner to disable the warnings.
